# Smart Alarm Mobile

Aplicación Android para gestionar alarmas inteligentes.

## Instalación rápida

1. Descarga el APK: `SmartAlarm-debug.apk`
2. Instala en tu dispositivo Android
3. Abre la app y comienza a crear alarmas

## Construcción

```bash
cd SmartAlarmApp
./gradlew assembleDebug
```

El APK generado está en: `SmartAlarmApp/app/build/outputs/apk/debug/app-debug.apk`

## Características

- Crear y gestionar alarmas
- Sonidos personalizables
- Notificaciones en tiempo real
